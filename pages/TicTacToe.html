Tic-Tac-Toe

This game was made as an assignment for my P2 Programming class. It was used as a teaching tool to better understand 2d arrays as well as string builders and printing to file.

The Features

Tic-Tac-Toe allows 2 players to play agains eachother as many times as they want. After each game, the final board is saved to a txt file. The game can be replayed within the console. X always starts.

The Code

This project was an introduction to 2d arrays and advanced StringBuilder techniques. A main focal point of the project was input validation; both the board needed to be checked for occupied spots and the user needed to enter an appropriat input.

The Filnal result

The game can be played any number of times with the winning round being saved to an included txt file. The menu is simple and comprehensible showing the user the main board and a score board. When the game is done, the user can play agin without needing to re-open the program.
