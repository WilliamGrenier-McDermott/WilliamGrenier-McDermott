The Ranch

This game was made as the final project for P1 programming class. It was made to recap all the things we learnt in the semester. It features one of my first attempts at OOP (Object Oriented Programming).

The Features

The Ranch is a single-player time-attack game where the player has 2 minutes to collect as manypoints as possible. You do this by moving your player icon (@) over a cow (C); this wouldIncrease your score by 1. After 2 minutes, the game stops, your total score is displayed, and youYou are prompted if you would like to play again.

The Code

This project demonstrates my understanding of class development, input handling, and object control. As my first attempt at object-oriented programming, the program calculates cow movement by assigning random delta values across each axis. In contrast, the player's position is updated through user input. The menus are built using StringBuilder constructs and string arrays, which are manipulated to achieve the correct formatting.

The Final Result

The game was finished in a couple versions, slowly adding different features and fixing bugs. In the end, the game works as a clean CLI style arcade game that can be simply run without using many resources.
